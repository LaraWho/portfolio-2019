Science Portfolio Site

Site link GitHub link

I created a CMS from scratch using React, MongoDB, Mongoose, Express, Styled Components, and React-Quill.

This is a blog site for my brother to handle the uploading and editing of the science articles he writes.

I used JSON Web Tokens for his user authentication. Once he's logged in he can add, edit, and delete all content. A non-logged in user can browse the site and see all content.
