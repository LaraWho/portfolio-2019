Portfolio

Science Portfolio Site

Site Link GitHub Link

Full stack application using React, MongoDB, Mongoose, Express, Styled Components, and React-Quill. My brother asked me for a portfolio site which he could handle the uploading and editing of the science articles he writes. I used JSON Web Tokens for his user authentication. Once he's logged in he can add, edit, and delete all content. A non-logged in user can browse the site and see all content.

Cat Chase

Site Link GitHub Link

A full-stack group project. We created this 2D collision detection game in React, using Javascript for all game logic, MongoDB and Mongoose to store and retrieve the player data and points. It detects which device you're on, as it needs the keyboard to play, and is responsive.
See if you can beat the high score!

Colour Curiosity

Site Link GitHub Link

A front-end project implementing the ColorTag Api for the main use of the app, which is to upload a URL for an image and see a list of the colours which are most prominent in that image. If you particularly love the colours of an image and want to use them to build a colour scheme for any occasion (wedding theme, decorating a room etc) then use my app! I used local storage to save images and colour schemes. You're also able to delete them. There's a little Easter Egg on the home page which will take you to a fun page which displays a random colour theme, from the colr.org API.

Time To Travel

Site Link GitHub Link

A Doctor Who themed wanderlust journal for travellers, built in React. I made all graphics from scratch in Figma, and animations are pure CSS. I used Bcrypt to safely store and hash passwords, and Nodemailer to notify the user of a successful registration. Awarded Best CSS in my cohort at DevMountain.
