Colour Curiosity

Site Link GitHub Link

A front-end project implementing an API. I chose the ColorTag Api for the main use of the app, which is to upload a URL for an image and see a list of the colours which are most prominent in that image. If you particularly love the colours of an image and want to use them to build a colour scheme for any occasion (wedding theme, decorating a room etc) then use my app! I used local storage to allow you to save images and colour schemes. You're also able to delete them. There's a little Easter Egg on the home page which will take you to a fun page which displays a random colour theme, from the colr.org API.
